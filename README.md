# jaywalking

## Introduction

A game created using [React](https://reactnative.dev) and [Recoil](https://recoiljs.org).

## Demo

![Demo](./public/Demo.gif)

## Setup

1. Install the project using npm:

```
$ cd jaywalking && npm install
```

2. Start the project:

```
$ npm start
```
